A full-stack application that provides REST APIs to manage a developer profile, skills, and projects, with a minimal frontend to search and display data.

## Features

- Create, read, and update developer profile
- Store skills and projects
- Search projects by skill
- Health check endpoint
- CORS-enabled API
- Seeded database with sample data
- Fully deployed frontend and backend

---

## Tech Stack

### Backend
- Node.js
- Express.js
- TypeScript
- Prisma ORM
- PostgreSQL
- CORS
- dotenv

### Frontend
- React / Next.js
- Fetch API

### Hosting
- Backend: Render
- Frontend: Vercel
- Database: PostgreSQL (Render)
